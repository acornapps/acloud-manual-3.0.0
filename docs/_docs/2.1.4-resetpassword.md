### 2.1.4 사용자 비밀번호 초기화

---

청지윈 사용자 비밀번호를 초기화 하는 기능이다.\(초기화 후 제공받는 초기 비밀번호는 'Pass0000'이다.\)

##### a\) 환경설정 → 사용자 → 사용자 액션 메뉴 → 사용자 비밀번호 초기화를 클릭한다.![](/assets/KR/3.0.0/2.1.4_1.png)
