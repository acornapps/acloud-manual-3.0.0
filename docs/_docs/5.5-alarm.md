## 5.5 알람

---

클러스터에 대한 이벤트를 알람으로 받을 수 있다.

**a\) 클러스터 **→** 클러스터 선택 →** **알람을 클릭한다.**![](/assets/KR/3.0.0/5.5_1.png)![](/assets/KR/3.0.0/5.5_2.png)

**b\) 알람 명 클릭 시 상세 내용을 확인 할 수 있다.**
