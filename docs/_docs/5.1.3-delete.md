### 5.1.3 클러스터 삭제

---

클러스터 정보를 삭제하는 기능이다. 해당 클러스터를 사용하고 있는 어플리케이션 맵이 있을 경우 삭제할 수 없다.

##### a\) 클러스터 → 클러스터 선택 → 등록관리 → 삭제버튼을 클릭한다.![](/assets/KR/3.0.0/5.1.3_1.png)![](/assets/KR/3.0.0/5.1.3_2.png)
